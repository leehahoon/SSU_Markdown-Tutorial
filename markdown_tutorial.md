## 1. 개요
## Markdown이란?
마크다운(markdown)은 2004년 [John Gruber](https://daringfireball.net/)가 만든 일반 텍스트 기반의 경량 마크업 언어입니다. 일반 텍스트로 서식이 있는 문서를 작성하는데 사용됩니다. HTML과 RTF 등 서식 문서로 쉽게 변환되기 때문에 응용 소프트웨어와 함께 배포되는 README 파일이나 온라인 게시물 등에 많이 사용됩니다.

## Markdown의 특징
마크다운은 [WYSIWYG](https://ko.wikipedia.org/wiki/위지위그) 방식과 다르게, 텍스트에 마크다운 구문을 추가하여 어떤 단어와 문장이 다르게 표시되어야하는지 나타냅니다. 예를 들어, 제목을 표시하려면 #을 추가합니다. 또는 문장을 굵게 만들려면 앞뒤에 별표 두 개를 추가합니다.

## Markdown의 쓰임
마크다운은 쉬운 방법으로 메모를 작성하고 여러 콘텐츠를 제작하고 활용할 수 있습니다. 웹사이트 제작, 문서 작업, 노트, 서적 출판, 프레젠테이션, 이메일, 협업 도구 등 여러곳에 사용되고 있습니다.

가장 유명한 활용 중 하나는 [Jekyll](https://www.markdownguide.org/tools/jekyll/) 입니다. 마크다운 파일을 가져와 HTML 웹사이트를 구축하는 인기있는 정적 사이트 생성기로 Github Pages에서 Jekyll로 생성한 웹사이트에 대해 무료 호스팅을 제공합니다. 자세한 사항은 [다음](https://www.markdownguide.org/tools/github-pages/)을 통해 찾아볼 수 있습니다.

## 2. 기본 문법
## 제목
제목을 만드려면 #을 추가합니다. HTML에서 `<h1>`, `<h2>` 태그와 유사하고, #의 개수가 `<h1>` 태그의 숫자를 의미합니다. 예를 들어, `<h3>`을 만들고 싶으면, "### 제목"을 입력하면 됩니다.

이때, #과 제목 사이에 공백을 두어야합니다.

|마크다운|HTML|출력|
|:---:|:---:|:---:|
|# 제목 1| \<h1>제목 1\</h1>|<h1>제목 1</h1>|
|## 제목 2|\<h2>제목 2\</h2>|<h2>제목 2</h2>|
|### 제목 3|\<h3>제목 3\</h3>|<h3>제목 3</h3>|
|#### 제목 4|\<h4>제목 4\</h4>|<h4>제목 4</h4>|
|##### 제목 5|\<h5>제목 5\</h5>|<h5>제목 5</h5>|
|###### 제목 6|\<h6>제목 6\</h6>|<h6>제목 6</h6>|

## 글자 강조
텍스트를 굵게 또는 기울임 꼴을 만들어 강조할 수 있습니다.
## 굵은 글씨체
텍스트를 굵게 표시하려면 단어 또는 구문 앞뒤에 두 개의 별표 또는 밑줄을 추가합니다. 만약 강조를 위해 단어 중간을 굵게 표시하려면 문자 주위에 공백없이 별표 두 개를 추가합니다.
|마크다운|HTML|출력|
|:---:|:---:|:---:|
|글씨를 \*\*굵게\*\* 만들어 보자.|글씨를 \<strong>굵게\</strong> 만들어 보자.|글씨를 **굵게** 만들어 보자.|
|글씨를 \_\_굵게\_\_ 만들어 보자.|글씨를 \<strong>굵게\</strong> 만들어 보자.|글씨를 __굵게__ 만들어 보자.|
|글씨가\*\*굵게\*\*변한다.|글씨가\<strong>굵게\</strong>변한다.|글씨가**굵게**변한다.|

<br><br>

## 기울임체
텍스트를 기울임 꼴로 표시하려면 단어 또는 구문 앞뒤에 별표 또는 밑줄을 추가합니다. 만약 강조를 위해 단어 중간을 기울임 꼴로 표시하려면 문자 주위에 공백없이 별표 하나를 추가합니다.
|마크다운|HTML|출력|
|:---:|:---:|:---:|
|글씨를 \*기울임체\* 로 해보자.|글씨를 \<em>기울임체\<em>로 해보자.|글씨를 *기울임체* 로 해보자.|
|글씨를 \_기울임체\_ 로 해보자.|글씨를 \<em>기울임체\<em>로 해보자.|글씨를 _기울임체_ 로 해보자.|
|글씨가\*기울임\*체로 변한다.|글씨가\<em>기울임\<em>체로 변한다.|글씨가*기울임*체로 변한다.|

<br><br>

## 굵은 기울임체
굵은 글씨체와 기울임체를 동시에 사용하려면 단어나 구문 앞뒤에 별표 또는 밑줄 세 개를 추가합니다. 강조를 위해 단어 중간을 표시하려면 문자 주위에 공백없이 별표 세 개를 추가합니다.
|마크다운|HTML|출력|
|:---:|:---:|:---:|
|글씨를 \*\*\*굵은 기울임체로\*\*\* 해보자.|글씨를 \<strong>\<em>굵은 기울임체\</strong>\</em>로 해보자.|글씨를 ***굵은 기울임체로*** 해보자.|
|글씨를 \_\_\_굵은 기울임체로\_\_\_ 해보자.|글씨를 \<strong>\<em>굵은 기울임체\</strong>\</em>로 해보자.|글씨를 ___굵은 기울임체로___ 해보자.|
|글씨를 \_\_\*굵은 기울임체로\_\_\* 해보자.|글씨를 \<strong>\<em>굵은 기울임체\</strong>\</em>로 해보자.|글씨를 __*굵은 기울임체로*__ 해보자.|

## 인용구
인용구를 만들려면 단락 앞에 '>' 를 추가하십시오.

**예제)**  
```
> 동해물과 백두산이 마르고 닳도록
```

**출력)**
> 동해물과 백두산이 마르고 닳도록

<br>
<br>

## 여러 단락이 있는 인용구
인용구는 여러 단락을 포함할 수 있습니다. '>' 단락 사이에 빈줄을 추가합니다.

**예제)**  
```
> 동해물과 백두산이 마르고 닳도록
>
> 하느님이 보우하사 우리나라 만세
```

**출력)**
> 동해물과 백두산이 마르고 닳도록
>
> 하느님이 보우하사 우리나라 만세

<br>
<br>

## 중첩 인용구
중첩하여 인용구를 사용할 수 있습니다. 중첩하려는 단락 앞에 '>>'를 추가합니다.

**예제)**  
```
> 동해물과 백두산이 마르고 닳도록
>
>> 하느님이 보우하사 우리나라 만세
```

**출력)**
> 동해물과 백두산이 마르고 닳도록
>
>> 하느님이 보우하사 우리나라 만세

<br>
<br>

## 다른 요소와 인용구
인용구는 다른 마크다운 형식의 요소를 포함할 수 있습니다.

**예제)**  
```
> ## 애국가  
> - 1절  
>  
> **동해물과** 백두산이 마르고 닳도록
```

**출력)**
> ## 애국가  
> - 1절  
>  
> **동해물과** 백두산이 마르고 닳도록

## 리스트
콘텐츠의 항목을 정렬된 목록과 정렬되지 않은 목록으로 구성할 수 있습니다.

## 순서 리스트

**예제1)**  
```
1. 첫번째 항목
2. 두번째 항목
3. 세번째 항목
4. 네번째 항목
```

**출력1)**
1. 첫번째 항목
2. 두번째 항목
3. 세번째 항목
4. 네번째 항목

**예제2)**  
```
1. 첫번째 항목
    1. 첫번째 들여쓰기 항목
3. 두번째 항목
    1. 두번째 들여쓰기 항목
```

**출력2)**
1. 첫번째 항목
    1. 첫번째 들여쓰기 항목
3. 두번째 항목
    1. 두번째 들여쓰기 항목

<br>
<br>

## 기호 리스트
순서가 아닌 기호 리스트를 만들려면 항목 앞에 '-', '*', 또는 '+' 기호를 추가합니다. 순서 리스트와 마찬가지로 중첩된 목록을 만들려면 들여쓰기를 합니다.

**예제)**  
```
- 첫번째 항목
    * First item
    * Second item
- 두번째 항목
    + First item
    + Second item
- 세번째 항목
- 네번째 항목 
```

**출력)** 
- 첫번째 항목
    * First item
    * Second item
- 두번째 항목
    + First item
    + Second item
- 세번째 항목
- 네번째 항목

<br>
<br>
